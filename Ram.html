Ram

Ram (Random Access Memory) is the computer's short-term memory. It is used to handle all active tasks and applications.


Different Ram Generations:

  • + DDR - Double Data Rate First Generation
  • + DDR2 - Double Data Rate Second Generation
  • + DDR3 - Double Data Rate Third Generation
  • + DDR4 - Double Data Rate Fourth Generation
